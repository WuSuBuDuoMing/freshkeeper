# 产品需求文档

## 项目概述

**项目名称**：AI 冰箱食材管理助手
**目标用户**：家庭主妇/主夫、健身人群、独居青年、健康饮食关注者
**核心价值**：减少食材浪费、提升烹饪效率、智能管理冰箱

## 功能模块

### 1. 首页仪表盘
- 食材总数、即将过期数、已过期数展示
- 今日待处理食材列表
- 今日推荐菜谱卡片
- 快捷入口：添加食材、过期提醒、菜谱、采购
- 本周浪费估算金额

### 2. 食材管理
- 9大分类：蔬菜、水果、肉类、海鲜、蛋奶、饮料、调料、主食、其他
- 4种状态：正常、即将过期、已过期、已用完
- 3种排序：过期时间、名称、添加时间
- 关键词搜索
- 标记已用完、删除操作

### 3. 食材添加/编辑
- 表单字段：名称、分类、数量、单位、存放位置、购买日期、过期日期、备注
- 表单校验：名称必填、过期日期必填、日期逻辑校验
- 保存后自动返回并刷新列表

### 4. 过期提醒
- 四级分类：今日过期、3天内、7天内、已过期
- 每项支持：标记用完、加入菜谱、加入采购、删除
- 计数徽标展示各类数量

### 5. 采购清单
- 手动添加采购项
- 从已用完食材快速添加
- 按分类分组展示
- 预计花费统计
- 已购买/待购买 Tab 切换

### 6. AI 菜谱推荐
- 基于冰箱现有食材匹配菜谱
- 优先推荐使用即将过期食材的菜谱
- 展示：已有食材、缺少食材、烹饪步骤、推荐理由
- 缺少食材一键加入采购清单

### 7. 一周菜单
- 自动生成 7 天早餐/午餐/晚餐
- 每天支持重新生成
- 每天支持一键加入采购清单
- 全周食材一键采购

### 8. 统计分析
- 食材分类占比柱状图
- 月度浪费统计
- 最常购买食材排行
- 省钱建议

### 9. 个人中心
- 暗黑模式切换
- 清除本地数据
- 版本信息
- 分享功能

## 非功能需求

- **性能**：页面首次加载 < 1s，列表滑动流畅
- **兼容性**：iOS/Android 微信客户端，基础库 3.3.4+
- **数据安全**：所有数据本地存储，不上传服务器
- **用户体验**：卡片式 UI，支持暗黑模式，操作反馈及时

## MVP 范围

当前版本为 MVP，使用 mock 数据，包含完整的 UI 和交互逻辑。后续可接入真实后端 API。
