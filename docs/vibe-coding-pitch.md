## 项目完整介绍

**AI 冰箱食材管理助手**是一款基于微信小程序的智能冰箱管理工具。

在日常生活中，我们常常面临这样的困扰：买了食材忘记吃，等到发现时已经过期；打开冰箱不知道今天做什么菜；去超市又忘了要买什么。

这款小程序通过以下功能解决这些痛点：

1. **食材管理**：像管理库存一样管理你的冰箱，支持 9 大分类、4 种状态
2. **智能过期提醒**：分级预警系统（今日/3天/7天/已过期），让每一份食材都不被浪费
3. **AI 菜谱推荐**：基于冰箱现有食材智能推荐菜谱，优先使用即将过期的食材
4. **一周菜单**：自动生成 7 天三餐建议，告别每天纠结吃什么
5. **采购清单**：从用完的食材快速生成采购清单，按分类分组，显示预计花费
6. **统计分析**：可视化展示食材分类占比、浪费趋势、省钱建议

**技术亮点**：
- 纯前端实现，无需后端服务
- Mock 数据引擎，50+ 种真实食材数据
- CSS 变量主题系统，支持暗黑模式
- 组件化架构，9 个复用组件
- Service-Utils-Storage 三层数据架构
